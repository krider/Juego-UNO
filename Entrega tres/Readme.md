# 🎮 CardUNO - Juego de Cartas UNO en Java

## Descripción
Implementación del juego de cartas UNO en Java aplicando programación orientada a objetos, refactorización y buenas prácticas de desarrollo de software.

## Características
- ✅ 2-4 jugadores (1 humano + hasta 3 bots)
- ✅ Cartas especiales: Salto, Reversa, Roba 2, Roba 4, Comodín
- ✅ Dirección dinámica del juego (horario/antihorario)
- ✅ Interfaz de consola con colores ANSI
- ✅ IA básica para jugadores simulados
- ✅ Manejo robusto de excepciones
- ✅ Reglas completas del UNO clásico

## Requisitos
- Java 8 o superior
- Terminal que soporte colores ANSI (Linux, macOS, o Windows con PowerShell)
- Git (para control de versiones)
